Glow Button Effect

How to use:


<button class="glow-button">Hover Me</button>
            

.glow-button {
    padding: 15px 30px;
    background-color: #2ecc71;
    color: white;
    border: none;
    border-radius: 5px;
    font-size: 18px;
    cursor: pointer;
    position: relative;
    z-index: 1;
    overflow: hidden;
    transition: 0.5s;
}

.glow-button:hover {
    color: white;
    box-shadow: 0 0 20px #2ecc71;
}

.glow-button:active {
    transform: scale(0.95);
}
