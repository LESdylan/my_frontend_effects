Fill Button Effect

How to use:


<button class="fill-button">Hover Me</button>
            

.fill-button {
    padding: 15px 30px;
    background-color: transparent;
    color: #9b59b6;
    border: 2px solid #9b59b6;
    border-radius: 5px;
    font-size: 18px;
    font-weight: bold;
    cursor: pointer;
    position: relative;
    z-index: 1;
    overflow: hidden;
    transition: color 0.4s ease-in-out;
}

.fill-button::before {
    content: '';
    position: absolute;
    top: 0;
    left: 0;
    width: 0;
    height: 100%;
    background-color: #9b59b6;
    z-index: -1;
    transition: width 0.4s ease-in-out;
}

.fill-button:hover {
    color: white;
}

.fill-button:hover::before {
    width: 100%;
}
