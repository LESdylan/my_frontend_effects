Neon Button Effect

How to use:


<button class="neon-button">NEON GLOW</button>
            

.neon-button {
  position: relative;
  display: inline-block;
  padding: 15px 30px;
  font-size: 24px;
  color: #00FFFC;
  font-weight: bold;
  text-transform: uppercase;
  background: transparent;
  border: 2px solid #00FFFC;
  border-radius: 4px;
  cursor: pointer;
  letter-spacing: 3px;
  overflow: hidden;
  transition: 0.2s;
  box-shadow: 
    0 0 5px #00FFFC,
    0 0 5px #00FFFC inset;
  text-shadow: 0 0 2px #00FFFC;
}

.neon-button:hover {
  background-color: #00FFFC;
  color: #111;
  box-shadow: 
    0 0 10px #00FFFC,
    0 0 30px #00FFFC,
    0 0 50px #00FFFC;
  text-shadow: none;
}

.neon-button:active {
  transform: scale(0.95);
}
