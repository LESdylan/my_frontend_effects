Gradient Wave Background Effect

An animated gradient background with a flowing wave-like movement that creates a dynamic and engaging backdrop for your content.

How to use:


<div class="gradient-wave-bg"></div>
                

.gradient-wave-bg {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background: linear-gradient(45deg, #6a11cb, #2575fc, #00a1ff, #39c1b6);
    background-size: 400% 400%;
    z-index: -1;
    animation: gradientWave 15s ease infinite;
}

@keyframes gradientWave {
    0% {
        background-position: 0% 50%;
    }
    50% {
        background-position: 100% 50%;
    }
    100% {
        background-position: 0% 50%;
    }
}
